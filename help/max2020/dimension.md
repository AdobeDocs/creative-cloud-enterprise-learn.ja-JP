# [!DNL Dimension]

![チュートリアルヒーロー画像](../assets/Dimenio.jpg)

魅力的な 3D のコンテンツを高品質のモデル、マテリアル、ライトを使用して作成できます。 [!DNL Dimension] では、ブランドの視覚化、イラスト、製品モックアップ、パッケージデザインなどのクリエイティブ作品を簡単に制作できます。
