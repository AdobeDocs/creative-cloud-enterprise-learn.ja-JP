# スタイルを使用してカスタムモデルを作成する

Fireflyでカスタムモデルを構築し、背景、カラー、色相など特定のスタイルを使用して、組織のブランドに新しい画像を作成する方法を説明します。 カスタムモデルを使用すると、関連性のあるパーソナライズされたコンテンツを大規模に作成できます。 Adobe Fireflyを試す [こちら](https://firefly.adobe.com/).
