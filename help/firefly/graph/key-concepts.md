# &#x200B;2. Fireflyグラフのキーコンセプト

Fireflyグラフの使用を開始する際に役立つ重要な概念について説明します。

## 節

ノードは、ワークフロー内で1つのステップ（1つのノード、1つのジョブ）を実行します。 ノードでは、画像の生成、マスクの適用、カラーのシフトなど、クリエイティブなアクションを1回実行できます。

![ノード](../assets/node.png){align="center"}

## ポート

ノード上の接続ポイント。 出力ポートはノードからデータを渡し、入力ポートはデータを受信します。 ポートの接続は、ワークフローにおけるデータの流れです。

![ポート](../assets/port.png){align="center"}

## Widget

テキストフィールド、ドロップダウン、スライダーなどのノードのインタラクティブコントロールでは、エディターで直接設定を構成できます。

![ウィジェット](../assets/widget.png){align="center"}

## 接続

コネクションは、2つのノード間で入力または出力を伝送します。 グラフは、ソース入力から最終出力まで、左から右に読み取られます。

![接続](../assets/connection.png){align="center"}

## グラフ

エディターで構築する完全なワークフロー。 グラフは、最終出力を生成するためにカンバス上に配置されたノードとコネクションで構成されています。

![グラフ](../assets/graph.png){align="center"}
