# Fireflyグラフテンプレートライブラリ

各テンプレートは、誰かが既に作成した完全なグラフです。 一つを開いて正確に動作を確認し、自分の入力を入れ替えて自分のものにします。

| テンプレート | 機能 | リンク |
|---|---|---|
| **Bullet-Time VFX Generator** | 箇条書きスタイルのVFXシーケンスを生成します。 | テンプレートを開く |
| **デザインシステムジェネレーター** | 再利用可能なデザインシステムアセットを作成します。 | テンプレートを開く |
| **Hero&#39;s Journey Story Builder** | ストーリーボードやキャンペーンアーク用に、物語のシーケンスとストーリー構造を構築します。 | テンプレートを開く |
| **カスタムリアルタイムシェーダーとブラシストローク** | カスタムリアルタイムシェーダーおよびブラシストロークエフェクトを作成します。 | テンプレートを開く |
| **スケッチして3Dに変換** | スケッチを3D回転ビューに変換します。 | テンプレートを開く |
| **ターゲットの色補正** | 部分的なカラー補正を元のデータを損なわずに適用します。 | テンプレートを開く |
| **一貫した顔写真** | グループ全体で一貫したヘッドショットのセットを生成します。 | テンプレートを開く |
| **エディトリアル写真+服の入れ替え** | 撮影現場で衣服を交換しながら、エディトリアル写真の外観を作成します。 | テンプレートを開く |
