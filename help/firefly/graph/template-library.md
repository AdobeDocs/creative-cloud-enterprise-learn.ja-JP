# &#x200B;5. Fireflyグラフテンプレートライブラリ

各テンプレートは、誰かが既に作成した完全なグラフです。 一つを開いて正確に動作を確認し、自分の入力を入れ替えて自分のものにします。

| テンプレート | 機能 | リンク |
|---|---|---|
| **Bullet-Time VFX Generator** | 箇条書きスタイルのVFXシーケンスを生成します。 | [テンプレートを開く](https://firefly.adobe.com/graph/edit/id/urn:aaid:sc:VA6C2:b6edd909-614b-5c49-a79c-6d4f2cc18d03) |
| **デザインシステムジェネレーター** | 再利用可能なデザインシステムアセットを作成します。 | [テンプレートを開く](https://firefly.adobe.com/graph/edit/id/urn:aaid:sc:VA6C2:5300d435-e46d-5691-ba9c-501a3ba6fb25) |
| **Hero&#39;s Journey Story Builder** | ストーリーボードやキャンペーンアーク用に、物語のシーケンスとストーリー構造を構築します。 | [テンプレートを開く](https://firefly.adobe.com/graph/edit/id/urn:aaid:sc:VA6C2:065565bb-a76c-5332-ac80-9d0eed4dfdb8) |
| **カスタムリアルタイムシェーダーとブラシストローク** | カスタムリアルタイムシェーダーおよびブラシストロークエフェクトを作成します。 | [テンプレートを開く](https://firefly.adobe.com/graph/edit/id/urn:aaid:sc:VA6C2:aeace815-1165-5a35-a7d1-13964d3b8268) |
| **スケッチして3Dに変換** | スケッチを3D回転ビューに変換します。 | [テンプレートを開く](https://firefly.adobe.com/graph/edit/id/urn:aaid:sc:VA6C2:27f2a0dd-42fc-53d1-9e53-2e387168a0f7) |
| **ターゲットの色補正** | 部分的なカラー補正を元のデータを損なわずに適用します。 | [テンプレートを開く](https://firefly.adobe.com/graph/edit/id/urn:aaid:sc:VA6C2:7e5eb791-6100-503f-ab49-881da9da3e09) |
| **一貫した顔写真** | グループ全体で一貫したヘッドショットのセットを生成します。 | [テンプレートを開く](https://firefly.adobe.com/graph/edit/id/urn:aaid:sc:VA6C2:3b3bdfc4-b316-5930-ba96-44f02a99e9a4) |
| **エディトリアル写真+服の入れ替え** | 撮影現場で衣服を交換しながら、エディトリアル写真の外観を作成します。 | [テンプレートを開く](https://firefly.adobe.com/graph/edit/id/urn:aaid:sc:VA6C2:c40f4d51-6579-52f0-9481-8c876e27e460) |
