# &#x200B;4. テンプレートのカスタマイズ

すべてのテンプレートには、サンプル入力と一般的なユースケースが付属しています。 このワークフローは、自分の概要、アセット、および結果を反映するだけで、*あなた*&#x200B;に適したワークフローになります。

1. **サンプルの入力を自分のアセットまたは概要に置き換えます。**
2. **プレースホルダテキストを使用しているノードまたはラベルの名前を変更します。**
3. **結果を信頼する前に、グラフを1回だけ完全に実行してください。**
4. **1つの変数を変更して再実行する**&#x200B;ことで、そのステップが何を制御しているかを正確に確認できます。
5. **カスタマイズしたバージョンを独自の名前で保存します。**
