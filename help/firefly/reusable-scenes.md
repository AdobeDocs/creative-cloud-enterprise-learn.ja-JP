# 商品の再利用可能なシーンの作成

ハンドバッグやアクセサリーなどの商品の再利用可能なシーンを作成する方法を説明します。 再利用可能なシーンを作成すると、季節や好きな時間にウェブサイトの商品を更新できるため、デジタル商品の販売が促進されます。 [こちら](https://firefly.adobe.com/?locale=jp)でAdobe Fireflyを試してください。
