# Text-to-template Gen AIの使用方法

数秒以内にテキストの説明からソーシャル投稿、ポスター、チラシ、カードの編集可能なテンプレートを生成する方法を説明します。 ダウンロードや共有を行う前に、デザインの編集、フォントの変更、プロジェクトのブランディングを行うことができます。
