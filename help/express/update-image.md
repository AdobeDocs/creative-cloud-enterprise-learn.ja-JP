# 財務のラウンドアップイメージの更新

ジェネレーティブフィルを使用して要素を置換したり、書体と色を一致させてテキストを変更したり、新しい画像の背景を削除したりすることで、静的なグラフィック画像をすばやく更新できます。 これは、金融処理などのグラフィックアセットをすばやく簡単に転用する方法です。
