# [!DNL Dimension]

高品質のモデル、マテリアル、照明を使用して、魅力的なコンテンツを3Dでより迅速に作成します。 [!DNL Dimension]を使用すると、ブランドの視覚エフェクト、イラスト、製品のモックアップ、パッケージのデザイン、その他のクリエイティブな作品を簡単に作成できます。
